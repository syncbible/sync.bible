Getting Started

Sync.Bible is a powerful online platform designed to help you study the Bible effectively. To get started:

  • Visit Sync.Bible and sign up for free.
  • Choose your preferred Bible translation.
  • Explore study tools like cross-references, commentaries, and search functions.

Features

Sync.Bible offers a range of features to enhance your Bible study experience:

  • Personalized study notes and bookmarks.

Features

This is a list of the main features of Javascripture with some explanation and context

  • Offline
    Javascripture should work offline if your brower supports it. It uses a service worker which has pretty good support these days.
  • Word tree
    Strongs dictionary defines roots for each word. Javascripture uses this data to also create "branches" for each word too. A branch is a word which has the current word as its root. This makes it easier to find related words.
  • Hovering
    When you hover over a word, all the other words with the same strongs number are highlighted.
